# Dashboard-NeuroTech


Welcome


## Frontend
```
cd frontend

yarn install

yarn start
```



## Backend
```
cd backend

python3 -m venv env
```
Add your env folder in .gitignore

Start the virtual environment using:
```
source env/bin/activate
```



To install all of the necessary dependencies in it use:

```
pip install -r requirements.txt

python webapp_backend.py
```
